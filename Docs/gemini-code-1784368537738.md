# 📝 Game Design Document: Project World Apocalypse (Rough Draft)

เกมแนว **2D Top-Down Roguelike/Roguelite** ในธีมวันสิ้นโลกขนาดยักษ์ ผสมผสานระบบสำรวจแผนที่แบบตารางเชื่อมต่อ และการต่อสู้ที่รวดเร็วท้าทาย

---

## 1. Core Gameplay & Controls (ระบบการเล่นหลักและการควบคุม)
*   **8-Directional Movement:** ตัวละครเคลื่อนที่และหลบหลีกได้ 8 ทิศทาง (เหนือ, ใต้, ออก, ตก และแนวทแยง) เพิ่มมิติการควบคุมที่ลื่นไหล
*   **Combat Styles:** รองรับทั้งการโจมตีระยะประชิด (Melee Hack & Slash) ตามทิศทางที่หัน และการยิงระยะไกล (Twin-stick Shooter) เล็งอิสระ 360 องศา
*   **Key Mechanic:** ระบบ **Dash / Dodge Roll (พุ่งหลบ)** 8 ทิศทาง พร้อมช่วงเวลาอมตะ (I-frame) เพื่อใช้หลบห่ากระสุนหรือฝูงศัตรู

---

## 2. World Map Structure: The 4,096 Grid (โครงสร้างโลก)
*   **Overworld Scale:** แผนที่โลกมีขนาด $64 \times 64$ ช่อง รวมทั้งหมด **4,096 ด่าน (Local Stages)** ที่ถูกสุ่มสร้างขึ้น (Procedural Generation) ตั้งแต่เริ่มสร้างโลก
*   **Dynamic Level Loading (4-Way Exit):** 
    *   เมื่อผู้เล่นอยู่ในด่านย่อย (Local Map) จะสามารถเดินสำรวจและต่อสู้ได้อย่างอิสระ
    *   ที่ขอบสนามของแต่ละด่านจะมี **ทางออก 4 ทิศทาง (เหนือ, ใต้, ออก, ตก)** 
    *   เมื่อเดินทะลุขอบจอในทิศใดทิศหนึ่ง ตัวเกมจะโหลดฉากถาวรของช่องถัดไปในตาราง $64 \times 64$ ทันที
*   **Fog of War:** เริ่มต้นแผนที่ใหญ่จะมืดสนิท ผู้เล่นต้องเดินเท้าสำรวจเพื่อเปิดแผนที่ไปทีละช่อง

---

## 3. Biomes & Terrain (ภูมิประเทศและไบโอม)
มหานครขนาดใหญ่ที่ล่มสลายและถูกธรรมชาติทวงคืน แบ่งออกเป็นภูมิประเทศหลักๆ ดังนี้:
*   **เมืองใหญ่ (Urban Ruins):** ซากตึกหักพัง ห้างสรรพสินค้า เต็มไปด้วยสิ่งกีดขวาง
*   **ถนนหลวง (Highways):** ทางยาวที่มีซากรถชนกัน เหมาะกับการใช้ยานพาหนะเดินทางไว แต่เสี่ยงโดนดักซุ่ม
*   **พื้นที่ป่า (Overgrown Forests):** ป่าทึบและพืชกลายพันธุ์ มีพุ่มไม้ให้ใช้ลอบเร้น
*   **ภูเขาสูง (Highlands):** เดินผ่านยากใช้เวลามาก แต่ใช้เป็นจุดยุทธศาสตร์เปิดหมอกบนแผนที่ใหญ่ได้กว้างขึ้น
*   **แม่น้ำ/เส้นทางน้ำ (Waterways):** อุปสรรคธรรมชาติที่ใช้แบ่งเขต หรือใช้หลบภัยจากศัตรูบางประเภท
*   **หมู่บ้านรอบนอก (Suburban Villages):** แหล่งชุมชนเก่าที่มีโอกาสพบทรัพยากรและของประทังชีวิตสูง

---

## 4. Enemy Factions (ฝ่ายศัตรูสามขั้ว)
ศัตรูจะกระจายตัวอยู่ตามไบโอมที่เหมาะสม และมีระบบ **Infighting (สู้กันเอง)** หากพวกมันเดินมาเจอกัน

| ฝ่ายศัตรู | ธีมเด่น | พฤติกรรม / Mechanic | ถิ่นที่อยู่หลัก |
| :--- | :--- | :--- | :--- |
| **1. ซอมบี้ (Zombie / Horde)** | Biological Horror | เน้นจำนวนมหาศาล วิ่งเข้าใส่เมื่อได้ยินเสียงหรือเห็นตัว | เมืองใหญ่, ถนน, หมู่บ้าน |
| **2. เอเลี่ยน (Alien / Hive)** | Cosmic Sci-Fi | ว่องไว พรางตัวได้ พ่นกรด/พิษ โจมตีแบบลอบเร้นเป็นฝูง | ป่าทึบ, ภูเขาสูง |
| **3. โรบอต (Robot / Machine)** | AI Run Amok | เกราะหนา อึด โจมตีระยะไกลด้วยปืนกล/เลเซอร์ มีโล่บาเรีย | ใจกลางเมือง, ศูนย์วิจัย, สถานีไฟฟ้า |

---

## 5. Roguelike Elements & Game Loop
*   **Randomized Spawn (สุ่มเกิด):** ทุกครั้งที่เริ่มเกมใหม่ (New Run) ผู้เล่นจะถูกสุ่มไปเกิดใน 1 จาก 4,096 ช่อง ทำให้ประสบการณ์การเล่นแต่ละรอบไม่ซ้ำกัน
*   **Starting Safe Zone:** ด่านแรกที่สุ่มเกิดจะมีจุดปลอดภัยเล็กๆ (เช่น ซากยานตก หรือห้องใต้ดิน) พร้อมอาวุธเริ่มต้นและเรดาร์มองเห็นไบโอมรอบตัว 1 ช่อง เพื่อให้ตั้งหลักและวางแผนเส้นทาง
*   **Permadeath:** หากตายในรอบนั้นจะต้องเริ่มใหม่ทั้งหมด แต่อาจมีระบบคะแนน/ทรัพยากรพิเศษที่ทิ้งไว้เพื่ออัปเกรดตัวละครถาวรในหน้าเมนูหลัก (Roguelite Progression)
