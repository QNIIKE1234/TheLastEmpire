01

Core Gameplay Loop

🗺️

สำรวจ (Exploration)

เดินผ่านประตูพอร์ทัล 4 ทิศบนแผนที่กริดที่สุ่มสร้างไดนามิก ทุกห้องมี Seed ของตัวเองทำให้รูปแบบไม่ซ้ำกัน

💀

เอาชีวิตรอด (Survival)

ความหิวลดตลอดเวลา วิ่งเร็วหิวเร็ว x2 ถ้าหิวสุดจะเริ่มกิน HP แทน ต้องหาขนมปัง/โพชั่นมากินให้ทัน

⚔️

ต่อสู้ (Combat)

ยิงปืนระยะไกลหรือฟันอาวุธประชิด ศัตรูมีหลายประเภท ฆ่าแล้วได้รับเงินและไอเทมดรอปตาม Loot Table

💰

เศรษฐกิจ (Economy)

ขายไอเทมเหลือใช้ให้พ่อค้า NPC ในราคา 30% แล้วซื้อปืน, กระสุน, หรือโพชั่นเตรียมลุยด่านต่อไป

04

Technical Highlights

🔄 Clean Scene Reloading

เมื่อกด Retry, ObjectPoolManager ดักจับ sceneLoaded event ทำลาย Objects เก่าทั้งหมดและสร้าง Pool ใหม่ ป้องกันบั๊ก AI เป้าหมายเสีย

📦 Recursive Resources Load

ใช้ Resources.LoadAll<T>("") ทำให้ ItemDatabase ค้นหาไฟล์ไอเทมได้ทุกโฟลเดอร์ใน Resources ไม่ติดปัญหาย้ายโฟลเดอร์

🎬 DOTween Screen Transition

ใช้ DOTween.To() ทำ Fade ดำ → วาร์ป → รอ 0.5s → Fade สว่าง แทน DOFade() เพื่อหลีกเลี่ยงปัญหา Extension ไม่ถูก Setup

🎯 Data-Driven Design

สถิติอาวุธ, ราคาไอเทม, และ Loot Table ล้วนแก้ไขจาก ScriptableObject ใน Inspector โดยไม่ต้องแก้ไขโค้ด

ระบบ XP, เลเวลอัพ, และนำแต้มไปอัพเกรดสถานะหลัก เช่น ความอึด, ดาเมจพื้นฐาน, ความเร็วแดช
